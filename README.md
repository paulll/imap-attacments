Скачивает все вложения за указанный период времени для заданных аккаунтов

## Зависимости:
- node.js
- npm

## Установка:
```
git clone https://github.com/paulll/imap-attacments
cd imap-attachments
npm i .
```

## Использование:
```
# в папке проекта


node . -i passwords.txt -o out -f 12.09.15 -t 12.10.16
```
